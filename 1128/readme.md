# 언리얼 엔진 5 기초 가이드 핵심 정리

## EP1. 언리얼 엔진 설치 및 실행

1.  **Epic Games Launcher 설치 및 로그인**:
    * Epic Games 웹사이트에서 런처를 다운로드하고, 실행 후 계정을 생성하거나 로그인합니다.
2.  **언리얼 엔진 설치**:
    * 런처 좌측 메뉴바에서 **Unreal Engine** > **Library**로 이동합니다.
    * `Install Engine`을 클릭하여 설치를 진행하며, `Option` 버튼을 눌러 필요한 추가 기능을 선택/제외할 수 있습니다.
3.  **기존 프로젝트 확인**:
    * 런처 **Library** 탭 하단의 **My Projects** 공간에서 이전에 만든 프로젝트들을 확인할 수 있습니다.

## EP2. 프로젝트 생성과 기본 가이드

1.  **새 프로젝트 생성**:
    * 런처에서 **Launch** 버튼을 눌러 **Unreal Project Browser**를 엽니다.
    * **GAMES** 항목을 포함한 좌측 메뉴에서 원하는 **템플릿**을 선택하고 프로젝트 명을 입력한 후 `Create`를 클릭합니다.
2.  **주요 템플릿 종류**:
    * **GAMES**: First Person, Third Person, Virtual Reality 등.
    * **FILM/VIDEO & LIVE EVENTS**: Virtual Production (VP), InCameraVFX 등.
    * **ARCHITECTURE**: Archvis, Design Configurator 등.
3.  **언리얼 엔진 화면 구성**:
    * **뷰포트(Viewport)**: 게임 화면 확인 및 물체 배치 공간.
    * **아웃라이너(Outliner)**: 뷰포트 내 물체들의 계층 구조 표시.
    * **디테일(Detail)**: 오브젝트/스크립트의 세부 항목 설정 창.
    * **콘텐츠 브라우저(Content Browser)**: 게임에 필요한 자료 저장 공간 (단축키: **Ctrl+Space**).
    * **액터 배치(Actor Batch)**: UE에서 제공하는 기초 오브젝트 모음.
4.  **레벨 생성 및 환경 설정**:
    * 콘텐츠 브라우저에서 **Level**을 생성합니다.
    * **환경 라이트 믹서** 창이나 **액터 배치**를 사용하여 **Sky Light** (간접광), **Directional Light** (태양광), **Sky Atmosphere** (대기), **Volumetric Cloud** (구름), **Exponential Height Fog** (안개) 등의 환경 요소를 간편하게 생성할 수 있습니다.

## EP3. 블루프린트 기초

1.  **스크립팅 종류**:
    * **레벨 블루프린트**: 상단 툴바에서 `Open Level Blueprint`를 클릭하여 열며, 주로 레벨 자체의 이벤트를 처리합니다.
    * **일반 블루프린트**: 콘텐츠 브라우저에서 우클릭 후 `Blueprint Class`를 선택하여 생성하며, **Actor** 클래스를 주로 사용합니다.
2.  **블루프린트 창 레이아웃**:
    * **상태창**: 함수, 변수, 매크로, 이벤트 등을 관리.
    * **이벤트 그래프**: 실제 블루프린트 노드를 연결하여 상황을 구현하는 화면.
    * **디테일**: 특정 함수나 이벤트의 상세 설정 변경 공간.

## EP4. C++ 개발 기초

1.  **IDLE(통합 개발 환경) 세팅**:
    * **Visual Studio**를 설치하고, Visual Studio Installer에서 **"C++을 사용한 게임 개발"** 항목을 반드시 설치해야 합니다.
    * UE5 프로젝트에서 **툴** > **Visual Studio 열기**를 통해 프로젝트와 연동합니다.
2.  **기본 C++ 클래스 주요 함수**:
    * `CoreMinimal.h`: 언리얼 엔진의 핵심 헤더 파일.
    * `BeginPlay()`: 오브젝트 생성 직후 **단 한 번** 실행되는 함수.
    * `Tick()`: 오브젝트 생성 이후 **계속** 실행되는 함수 (매 프레임).

## EP5. 샘플 & 애셋 사용법

1.  **샘플 프로젝트 불러오기**:
    * Epic Games 런처의 **샘플** 항목에서 원하는 콘텐츠를 선택합니다.
    * **내 라이브러리 추가** 후 **프로젝트 생성**을 눌러 다운로드 및 프로젝트를 생성합니다.
2.  **무료 애셋 가져오기**:
    * 런처 좌측 메뉴의 **Fab** 항목(또는 Marketplace)으로 이동합니다.
    * 가격 필터를 **무료**로 설정하여 원하는 애셋을 찾습니다.
    * 다운로드 후 사용할 프로젝트를 선택하여 추가하면, 해당 애셋이 프로젝트의 콘텐츠 폴더에 추가됩니다.
